Come creare un tamplate per i post

di Simone Castelluccio - pubblicato il 23/01/2024

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando le parti comuni da applicare a tutte le pagine

Le varie aree saranno quindi suddivise in:

  • Contenuti di instestazione del tag <header>
  • Contenuti di instestazione del tag <main> (corpo principale)
  • Contenuti di instestazione del tag <footer>

Formattazione del codice

Il testo dell'articolo dovrà essere formato per identificare tutte le parti da selezionare con CSS. Perciò i passi saranno:

  1. Proggettare su carta il markup finale, assegnando ad ogni specifico test il tag appropriato
  2. Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  3. Applicare gli stili CSS

L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale.
